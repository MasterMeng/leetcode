# 113 路径总和 II  

## 题目描述  

给定一个二叉树和一个目标和，找到所有从根节点到叶子节点路径总和等于给定目标和的路径。

说明: 叶子节点是指没有子节点的节点。

## 难度

中等  

## 题解  

采用深度搜索优先，遍历每条从根节点到叶子节点的路径。遍历到叶子节点，且此路径和满足条件，则该路径即为一条满足要求的路径。
